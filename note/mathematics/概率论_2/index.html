# 二项分布（Binomial distribution）

指在只有两个结果的nnn次独立的伯努利试验中，所期望的结果出现次数的概率，记作X∼B(n,p)X \sim B(n,p)X∼B(n,p)。nnn次试验中正好得到kkk次成功的概率由**概率质量函数（Probability Mass Function, PMF）**给出：

P{X=k}=Cnkpk(1−p)n−kP\{X=k\} = C_n^k p^k(1-p)^{n-k} P{X=k}=Cnk​pk(1−p)n−k

概率质量函数是离散随机变量在各特定取值上的概率。

其中二项式系数：Cnk=n!k!(n−k)!C_n^k =\frac{n!}{k!(n-k)!}Cnk​=k!(n−k)!n!​

二项分布的期望值和方差：等于每次单独试验的期望和方差的和（独立的伯努利试验）

E(X)=npD(X)=np(1−p){\rm E} (X) = np \\ {\rm D}(X) = np(1-p) E(X)=npD(X)=np(1−p)

# 正态分布（Normal distribution）

也称为高斯分布（Gaussian Distribution），是常见的连续概率分布。
