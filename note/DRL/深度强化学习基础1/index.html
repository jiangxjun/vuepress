# 判别模型和生成模型

​ 深度学习模型大致分为两类：判别模型（Discriminative Models）和生成模型（Generative Models）。

​ 判别模型用于建模条件概率p(y∣x)p(\pmb{y}|\pmb{x})p(yy∣xx)，其中x\pmb{x}xx代表输入数据，y\pmb{y}yy是输出目标。判别模型基于输入，预测相应的标签y\pmb{y}yy。

判别模型大多应用于需要判断的任务，如分类任务，回归任务。

​ 分类任务中，模型需要根据输入数据从备选类别中选择正确的目标类别，典型包括二分类、多标签分类等。

​ 很多情况下，一个分类模型并不直接指定目标类别，而是给每一个备选类别计算一个概率，这主要是为了便于在训练阶段对模型进行优化。深度学习在图像分类、文本分类任务上取得很大成功。

​ 分类任务的输出均为离散的类别标签。

​ 回归任务的输出是连续的数值，如利用过去的交通数据预测未来一段时间内的车速[1][2]。

​ 生成模型用于建模联合概率p(x,y)p(\pmb{x},\pmb{y})p(xx,yy)。生成模型通常对可观测数据的分布进行建模，达到生成可观测数据的目的。比如生成对抗网络（Generative Adversarial Networks, GANs）（用于生成图像、重构图像和对图像去噪）[3]。

​ DL技术和可观测数据的分布没有显式关系，DL技术更关注生成的样本和可观测的真实样本之间的相似程度。朴素贝叶斯（Naive Bayes）[4]的生成模型也用于分类任务[5]。

​ 判别模型关注的是哪一个标签更适合可观测数据，生成模型尝试建模可观察数据的分布。

举例说明：

​ 1. 朴素贝叶斯对似然概率（Likelihood）p(y∣x)p(y|x)p(y∣x)建模，即可观测数据在给定标签情况下的条件概率。（判别模型）

​ 2. 当学习了联合概率分布p(x,y)p(x,y)p(x,y)之后即可学会判别，给定观测输入xxx，输出目标为1的概率为p(y=1∣x)=p(x,y=1)p(x)p(y=1|x) = \frac{p(x,y=1)}{p(x)}p(y=1∣x)=p(x)p(x,y=1)​。（生成模型）

​ 大多数深度神经网络是判别模型，很多生成类任务在具体实现中都可以简化为分类或者回归问题。
