HTML（Hyper Text Markup Language）超文本标记语言，是一种结构化Web网页及其内容的标记语言。

实例：

<!DOCTYPE html>   <!--声明文档的类型 -->
<html>    <!--HTML的根元素，结束标志</html> -->
    <head>   <!--包含文档的元数据（metadata），如<meta charset="utf-8">定义网页编码格式 -->
        <title>页面标题(w3cschool.cn)</title>  <!--定义文档的标题 -->
    </head>
    <body>                <!--定义文档的主体，网页可见的页面内容，有结束标志 -->
        <h1>我的第一个标题</h1> <!--作为一个标题使用 -->
        <p>我的第一个段落。</p> <!--作为一个段落显示 -->
    </body>
</html>

# div标签

<div>标签本身不代表任何东西，用于标记区域，网页中表示块、区域的意思。如样式化（使用class和id属性），用不同的语言（lang属性）标记html文档的某个部分或一个分隔区块。

<div>常用于组合块级元素，便通过CSS来对这些元素进行格式化。

<div>元素通常与CSS一起使用，布局网页

<!--文档中的一个区域显示为蓝色-->
<div style="color:#0000FF">
  <h3>这是一个在 div 元素中的标题。</h3>
  <p>这是一个在 div 元素中的文本。</p>
</div>

# class属性

class属性定义元素的类名，指向样式表的类。html中类名是区分大小写的。

class给块添加一些属性，并可以给属性起名字，class=xxx。

# id属性

id属性规定html元素的唯一的id，在html文档中必须是唯一的。id属性可用作链接锚（link anchor），通过JS或者CSS为带有指定id的元素改变或添加样式。

# id和class对比

id具有唯一性，在一个网页中同一个命名只能使用一次。

class命名的类可以在一个网页中使用无数次。

在css样式表中书写时，id选择符前面应该加前缀符号‘#’，而class选择符前面应该加前缀符号‘.’

class作为一个样式，先定义好，可以应用到任何结构和内容上（可复用）；id作为元素的标签，用于区分不同结构和内容（像名字），id是先找到结构/内容，在找它定义的样式；

class更多应用在文字版块和页面修饰中，id更多应用在布局和设计包含块、包含框中。

# span标签

<span>元素是无语义的行内元素，对元素进行分组，以不同的样式显示。

<span>元素可以应用class或id属性，class用于元素组（一类的元素），id用于标识单独唯一的元素。

<span>包含的文本，可以使用CSS对它定义样式，或者使用JS进行操作。

<p>我有 <span style="color:blue;font-weight:bold">蓝色</span> 的眼睛。</p>

# style标签

<style>标签包含了html文档的样式信息，默认在该元素内写入的样式指令是CSS。

一个html文档包含多个<style>标签。

需要链接外部样式表，使用<link>标签。
